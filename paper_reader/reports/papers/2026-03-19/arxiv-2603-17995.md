# LoST: Level of Semantics Tokenization for 3D Shapes

## Research Questions

- How can 3D shapes be tokenized by semantic salience rather than geometric level-of-detail to enable efficient autoregressive generation?
- How can semantic supervision be provided for 3D shape tokenization without direct access to semantic labels?
- Can semantically-ordered tokens enable high-quality 3D generation with dramatically fewer tokens than geometric hierarchies?

## How This Paper Solves It

Current 3D tokenization methods use geometric level-of-detail (LoD) hierarchies that require many tokens to represent basic shapes and produce unusable early prefixes. This paper proposes ordering tokens by semantic salience instead - early tokens capture the core semantic identity (e.g., 'chair') while later tokens add instance-specific details. The key insight is using Relational Inter-Distance Alignment (RIDA) to train a semantic extractor that aligns 3D shape relationships with DINO image feature relationships, providing semantic supervision without needing explicit 3D semantic labels.

## Key Contributions

- Level-of-Semantics Tokenization (LoST) that orders 3D shape tokens by semantic salience for prefix-decodable representations
- Relational Inter-Distance Alignment (RIDA) loss for aligning 3D triplane latent space with DINO semantic feature space
- SOTA 3D reconstruction and autoregressive generation using only 0.1%-10% of tokens needed by prior methods
- Demonstration that semantic tokenization enables downstream tasks like semantic shape retrieval
- Extension to multiple 3D representations including Direct3D triplanes and TRELLIS latents

## Key Formulas

### Inter-Instance Rank Distillation Loss

$$
L_{rank} := E_{z_i \in B}[||\tilde{c}_i^s - \tilde{c}_i^t||_2^2]
$$

Aligns relative similarity rankings between student (3D) and teacher (DINO) spaces. \tilde{c}_i^s and \tilde{c}_i^t are z-scored cosine similarity vectors that remove modality-specific means/scales, making the loss invariant to absolute similarity values while preserving neighborhood rankings.

### RIDA Combined Loss

$$
L_{RIDA} := \lambda_g L_{global} + \lambda_r L_{rank} + \lambda_s L_{spatial}
$$

Total semantic alignment objective combining global contrastive loss (L_global), inter-instance rank distillation (L_rank), and spatial structure distillation (L_spatial). Used to train the semantic extractor that maps 3D triplanes to semantically-structured features aligned with DINO space.

## Limitations of the Proposed Solution

- Relies on VAE triplane representation, limiting extension to other 3D formats without adaptation
- Uses diffusion decoder which increases computational cost compared to pure autoregressive decoding
- Few-token regime still exhibits artifacts compared to full-length decodes
- AR generator uses fixed target length rather than variable-length with EOS tokens for complexity-aware stopping

## Conclusions

- LoST achieves SOTA reconstruction with 31.649 FID and 0.731 DINO similarity using only 1 token vs 341.174 FID and 0.382 DINO for OctGPT with 219 tokens
- Autoregressive generation achieves 34.251 FID and 0.758 DINO similarity using 128 tokens, outperforming ShapeLLM-Omni (48.702 FID, 0.680 DINO) with 1024 tokens
- Semantic retrieval using RIDA features achieves 44.28% mAP@3 vs 16.42% for raw triplane features on in-distribution data
- Method generalizes across representations, showing consistent improvements on both Direct3D triplanes and TRELLIS latents

## General Limitations

- Relies on VAE triplane representation, limiting extension to other 3D formats without adaptation
- Uses diffusion decoder which increases computational cost compared to pure autoregressive decoding
- Few-token regime still exhibits artifacts compared to full-length decodes
- AR generator uses fixed target length rather than variable-length with EOS tokens for complexity-aware stopping

## Abstract

Tokenization is a fundamental technique in the generative modeling of various modalities. In particular, it plays a critical role in autoregressive (AR) models, which have recently emerged as a compelling option for 3D generation. However, optimal tokenization of 3D shapes remains an open question. State-of-the-art (SOTA) methods primarily rely on geometric level-of-detail (LoD) hierarchies, originally designed for rendering and compression. These spatial hierarchies are often token-inefficient and lack semantic coherence for AR modeling. We propose Level-of-Semantics Tokenization (LoST), which orders tokens by semantic salience, such that early prefixes decode into complete, plausible shapes that possess principal semantics, while subsequent tokens refine instance-specific geometric and semantic details. To train LoST, we introduce Relational Inter-Distance Alignment (RIDA), a novel 3D semantic alignment loss that aligns the relational structure of the 3D shape latent space with that of the semantic DINO feature space. Experiments show that LoST achieves SOTA reconstruction, surpassing previous LoD-based 3D shape tokenizers by large margins on both geometric and semantic reconstruction metrics. Moreover, LoST achieves efficient, high-quality AR 3D generation and enables downstream tasks like semantic retrieval, while using only 0.1%-10% of the tokens needed by prior AR models.
